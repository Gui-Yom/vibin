# Vibin

For when your last 2 brain cells need to vibe together at 4am.

![cat vibing](cat.gif)

## Build

```
cargo build --release
```

## Controls

- The window is freely movable by dragging it.
- Right-click to play next item
- Mouse wheel to adjust volume
- Middle mouse button to close it
